# By Sandrine Couture 🧵

Site vitrine professionnel pour **By Sandrine Couture**, artisan brodeur en Normandie (Robertot).
Ce projet met en avant le savoir-faire artisanal à travers une galerie optimisée et un système de devis premium.

## 🚀 Fonctionnalités

- **Galerie de Réalisations** : +50 images optimisées (WebP) avec filtrage par catégorie (Entreprises, Particuliers, Bébé, Accessoires).
- **Système de Devis Premium** : Formulaire adaptatif (Pro/Particulier) avec génération automatique de messages structurés pour WhatsApp.
- **Identité Visuelle** : Design moderne, épuré et responsive, intégrant la nouvelle charte graphique.
- **SEO & Performance** : Images converties et renommées pour le référencement, score de performance optimisé.
- **Pages Légales** : Mentions légales, CGV et Politique de Confidentialité intégrées.

## 🛠 Stack Technique

- **Framework** : Next.js 15+ (App Router)
- **Styling** : Tailwind CSS
- **Icônes** : Lucide React
- **Optimisation** : Sharp / Sips (Traitement d'images)
- **Déploiement** : Vercel

## 📦 Installation & Développement

1.  **Cloner le projet** :
    ```bash
    git clone https://github.com/Creativityliberty/sandrinecouture.git
    cd sandrinecouture
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Lancer le serveur de développement** :
    ```bash
    npm run dev
    ```

## 🛠 Scripts Utilitaires

Des scripts personnalisés sont disponibles dans le dossier `/scripts` :
- `process_images.sh` : Conversion massive des photos (HEIC/JPG) vers WebP optimisé.
- `rename_seo.sh` : Renommage automatique des fichiers pour le SEO.
